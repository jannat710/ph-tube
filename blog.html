1. Discuss the scope of var, let, and const

var:

It creates a function scoped variable which can be reassigned or redeclared .

let:

It creates a block-scoped variable which can be reassigned but can not be redeclared .

const:

It also creates a block-scoped variable but can not be reassigned or redeclared .

2. Tell us the use cases of null and undefined

Null

  1. Variable that is not initialized will give undefined
  2. Function with no return
  3. Parameter that is not passed will be undefined
  4. If return has nothing on the right side will return undefined
  5. Property that doesn't exists on an object will give you undefined
  6. Accessing array elements outside of the index range
  7. Deleting an element inside an array
  8. Set a value directly to undefined

Null

  1. Null is a special value that represents a missing object.
  2. Could be assign
  3. Type of object
  4. Variables assigned with literal null
  5. Null is falsy, meaning it evaluates to false in a boolean context.
  6. To check if a value is null , you use the strict equality operator ===
  7. Type of null return an object
  8. If the function returns null instead of an object, you'll get a TypeError
